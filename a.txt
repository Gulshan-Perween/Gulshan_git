Git & GitHub
Git is a version control system
GitHub is remote repository

hello changes
git init = to create git folder

git config user.name 'username' = to create new user
git config user.email 'email' = to create user email
 git status = to check the status
git add "filename" = to add new to file
git commit -m"message"   = message related to changes
git remote -v = to check number of the origin 
git branch =  to check the branch on which we are working
git checkout -b branchName = to create new branch
git checkout branchName = to come back to the first branch
